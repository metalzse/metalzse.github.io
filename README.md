# 兒童學習遊戲

這個專案是純前端的兒童學習遊戲入口頁，可直接部署到 GitHub Pages。

## 學習功能

- `ZhuyinAdventure/`：注音冒險故事，透過閱讀故事、選擇路線與收集道具練習注音閱讀。
- `ZhuYingFlashCard/`：注音練習卡，隨機顯示注音符號，由使用者唸出發音並由旁人判斷答對或答錯。

## 主選單工具

- `匯入學會的字`：在最上層選單匯入先前輸出的學會字清單，合併到共用學會字庫，供各學習功能使用。
- `清除目前記錄`：在使用者確認後，清除共用學會字庫、注音冒險進度與背包、注音練習卡統計。

## 使用方式

直接開啟根目錄的 `index.html`，再選擇要使用的學習功能或主選單工具。

所有內容都是純前端功能，不需要安裝套件或啟動後端。
